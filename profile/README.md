## About Tradable

Tradable is a retail-focused algorithmic trading platform that enables users to build, automate, and monetise trading strategies without writing code.

We combine:

- A no-code trading bot builder  
- A community strategy marketplace  
- AI-driven analytics and automation  
- Tokenised infrastructure powering next-gen on-chain AI  

Our mission is simple:

> Give everyday traders access to infrastructure that was previously reserved for institutions.

---

## Products

### Tradable Platform

A browser-based progressive web app that enables:

- Strategy creation via a no-code interface  
- Automated execution via exchange APIs  
- Portfolio aggregation  
- Marketplace strategy subscriptions  
- Data-driven optimisation tools  

Tradable is currently in public beta.

## Security & Responsibility

Tradable is a non-custodial platform.

We do not:
- Custody user funds  
- Provide financial advice  
- Execute discretionary trading  

All trading automation is controlled by the user.

If you believe you have discovered a security vulnerability, please contact us via our website rather than opening a public issue.
